# study-planner-app
A simple study planner app with timetable, daily goals and progress tracking for students.
# Study Planner App

## About
Study Planner ek simple aur powerful Android app hai
jo students ko apni padhai organize karne me help karta hai.

Is app me user:
- Daily timetable bana sakta hai
- Date-wise study planning kar sakta hai
- Roz ke study goals set kar sakta hai
- Apni progress track kar sakta hai

---

## Features

### 1. Timetable
- Daily aur weekly timetable
- Subject + time set karna
- Edit aur delete option
- Offline support

### 2. Daily Goals
- Roz ke study goals add karna
- Goals complete mark karna
- Pending goals dekhna

### 3. Progress Tracking
- Completed goals count
- Progress percentage
- Progress bar
- Daily performance view
